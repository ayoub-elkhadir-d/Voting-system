## Typical User Flows

```
FLOW 1: Room Owner Setup
═══════════════════════════════════════════════════════════════════════════════

  1. Register Account
       ↓
  2. Login
       ↓
  3. Create Room (set visibility: public/private, member limit)
       ↓
  4. Create Topic (add choices, set voting method)
       ↓
  5. Start Room (broadcast to participants)
       ↓
  6. Start Topic (activate voting)
       ↓
  7. View Admin Panel (monitor participants & votes)
       ↓
  8. Approve Members (if private room)
       ↓
  9. Stop Topic (end voting)
       ↓
  10. View Vote Statistics (analyze results)


FLOW 2: Participant Voting
═══════════════════════════════════════════════════════════════════════════════

  1. Register Account (if new user)
       ↓
  2. Login
       ↓
  3. Join Room (enter 6-digit code)
       ↓
  4. Enter Username (set display name)
       ↓
  5. Wait for Approval (if private room)
       ↓
  6. Cast Vote (select option(s))
       ↓
  7. View Live Results (see vote counts)
       ↓
  8. Leave Room (exit when done)


FLOW 3: Account Recovery
═══════════════════════════════════════════════════════════════════════════════

  1. Login (forgot password)
       ↓
  2. Request Password Reset
       ↓
  3. Check email for magic link
       ↓
  4. Verify link (auto-login)
       ↓
  5. Login with new password
```
