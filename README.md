
#  Voting System

A simple and secure ** Voting System** that allows users to vote for candidates and view real-time results.
This project demonstrates core concepts of full-stack development including authentication, database handling, and dynamic result updates.

---

##  Features

*  User registration and login system
*  Vote for available candidates
*  Real-time vote counting and result display
*  Basic security to prevent multiple voting
*  Admin panel to manage candidates
*  Simple and responsive user interface

## ⚙️ How It Works

1. User registers or logs in
2. System displays list of candidates
3. User selects a candidate and submits a vote
4. Vote is stored in the database
5. Results are updated and displayed in real-time or on refresh
